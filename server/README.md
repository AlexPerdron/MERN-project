### Deloyment
### run yarn to install dependencies
```bash
yarn
```
### start the client
```bash
yarn run client
```
### start the server
```bash
yarn run server
```
### Description
This is a personal fullstack project which will provide cv, personal infomation, achievement, etc.

### Technology
- NodeJS 14+
- Yarn package manager
- Express for server side
- React for frontend
- MongoDB as database
